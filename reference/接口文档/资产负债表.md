# 资产负债表

**文档ID**: 36
**接口**: balancesheet

## 接口描述

获取上市公司资产负债表数据。用户需要至少2000积分才可以调取。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| start_date | str | N | 公告日开始日期 |
| end_date | str | N | 公告日结束日期 |
| period | str | N | 报告期 |
| comp_type | str | N | 公司类型：1一般工商业 2银行 3保险 4证券 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| cip | float | **在建工程** |
| fix_assets | float | 固定资产 |
| total_assets | float | 资产总计 |
| total_liab | float | 负债合计 |
| total_hldr_eqy_exc_min_int | float | 股东权益合计 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.balancesheet(
    ts_code='600000.SH',
    start_date='20240101',
    end_date='20241231'
)

print("在建工程数据：")
print(df[['ts_code', 'end_date', 'cip', 'fix_assets', 'total_assets']].head())
```

## 重要字段说明

- **cip (在建工程)**：公司正在建设的工程项目价值，是资本开支的重要参考指标
- **fix_assets (固定资产)**：公司已建成并投入使用的固定资产净值
