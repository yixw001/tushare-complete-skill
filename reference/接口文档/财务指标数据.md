# 财务指标数据

**文档ID**: 33
**接口**: fina_indicator

## 接口描述

获取上市公司财务指标数据。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| start_date | str | N | 公告日开始日期 |
| end_date | str | N | 公告日结束日期 |
| period | str | N | 报告期 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| roe | float | **净资产收益率** |
| roa | float | 总资产收益率 |
| net_profit_margin | float | 销售净利率 |
| gross_margin | float | 销售毛利率 |
| debt_to_assets | float | 资产负债率 |
| current_ratio | float | 流动比率 |
| quick_ratio | float | 速动比率 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.fina_indicator(
    ts_code='600000.SH',
    start_date='20240101',
    end_date='20241231'
)
print(df[['ts_code', 'end_date', 'roe', 'debt_to_assets']].head())
```

## 重要字段说明

- **roe (净资产收益率)**：衡量公司盈利能力的重要指标
- **debt_to_assets (资产负债率)**：反映公司财务风险
