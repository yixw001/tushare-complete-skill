# 指数基本信息

**文档ID**: 75
**接口**: index_basic

## 接口描述

获取指数基本信息。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| market | str | N | 市场代码：SSE上交所SZSE深交所 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS指数代码 |
| name | str | 指数名称 |
| market | str | 市场代码 |
| publisher | str | 发布机构 |
| index_type | str | 指数类型 |
| category | str | 类别 |
| base_date | str | 基期 |
| base_point | float | 基点 |
| list_date | str | 上市日期 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.index_basic(market='SSE')
print(df.head())
```
