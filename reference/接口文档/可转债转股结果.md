## 可转债转股结果

接口：cb_conversion描述：获取可转债转股结果数据限量：单次最大2000，可多次提取，总量不限制积分：用户需要至少2000积分才可以调取，5000积分以上频次相对较高，积分越多权限越大，具体请参阅积分获取办法

接口示例

```

import tushare as ts

pro = ts.pro_api()

#获取可转债转股结果数据
df = pro.cb_conversion(start_date='20190101', end_date='20201231', fields='ts_code,ann_date,conv_vol,conv_amount,conv_price,conv_ratio')

```
