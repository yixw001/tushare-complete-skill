# 现金流量表

**文档ID**: 44
**接口**: cashflow

## 接口描述

获取上市公司现金流量表数据。用户需要至少2000积分才可以调取。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| start_date | str | N | 公告日开始日期 |
| end_date | str | N | 公告日结束日期 |
| period | str | N | 报告期 |
| comp_type | str | N | 公司类型：1一般工商业 2银行 3保险 4证券 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| c_pay_acq_const_fiolta | float | **购建固定资产、无形资产和其他长期资产支付的现金（资本开支）** |
| n_cashflow_act | float | 经营活动产生的现金流量净额 |
| n_cashflow_inv_act | float | 投资活动产生的现金流量净额 |
| n_cashflow_fin_act | float | 筹资活动产生的现金流量净额 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.cashflow(
    ts_code='600000.SH',
    start_date='20240101',
    end_date='20241231'
)

print(df[['ts_code', 'end_date', 'c_pay_acq_const_fiolta', 'n_cashflow_act']].head())
```

## 重要字段说明

### 资本开支（c_pay_acq_const_fiolta）
购建固定资产、无形资产和其他长期资产支付的现金，反映企业资本开支情况。

**重要性**: ⭐⭐⭐
- 用于计算自由现金流
- 评估企业扩张和投资活动
- 分析企业再投资能力
