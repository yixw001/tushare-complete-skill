# 上市公司公告

**文档ID**: 176
**接口**: anns_d

## 接口描述

获取全量公告数据，提供PDF下载URL。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| ann_date | str | N | 公告日期（YYYYMMDD） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ann_date | str | 公告日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| title | str | 标题 |
| url | str | URL原文下载链接 |
| rec_time | str | 发布时间 |

## 代码示例

```python
import tushare as ts
pro = ts.pro_api('your_token')
df = pro.anns_d(ann_date='20230621')
print(df.head())
```
