# 业绩快报

**文档ID**: 46
**接口**: express

## 接口描述

获取上市公司业绩快报。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| ann_date | str | N | 公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| revenue | float | 营业收入（元） |
| operate_profit | float | 营业利润（元） |
| total_profit | float | 利润总额（元） |
| n_income | float | 净利润（元） |
| total_assets | float | 总资产（元） |
| diluted_eps | float | 每股收益（摊薄）（元） |
| diluted_roe | float | 净资产收益率（摊薄）（%） |

## 代码示例

```python
import tushare as ts
pro = ts.pro_api('your_token')
df = pro.express(ts_code='600000.SH', start_date='20240101', end_date='20241231')
print(df.head())
```
