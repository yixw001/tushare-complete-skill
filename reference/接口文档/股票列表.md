# 股票列表

**文档ID**: 25
**接口**: stock_basic

## 接口描述

获取股票基本信息，包含上市A股的基本信息。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| list_status | str | N | 上市状态：L上市D退市P暂停上市 |
| exchange | str | N | 交易所：SSE上交所SZSE深交所 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| symbol | str | 股票简称 |
| name | str | 股票名称 |
| area | str | 所在地 |
| industry | str | 所属行业 |
| list_date | str | 上市日期 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.stock_basic(list_status='L')
print(df.head())
```
