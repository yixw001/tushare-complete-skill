## 国债收益率曲线利率（日频）

接口：us_tycr描述：获取美国每日国债收益率曲线利率限量：单次最大可获取2000条数据权限：用户积累120积分可以使用，积分越高频次越高。具体请参阅积分获取办法

接口调用

```

pro = ts.pro_api()

df = pro.us_tycr(start_date='20180101', end_date='20200327')


#获取1月期和1年期数据
df = pro.us_tycr(start_date='20180101', end_date='20200327', fields='m1,y1')

```
