# 上海黄金基础信息

**文档ID**: 284
**接口**: sge_basic

## 接口描述

获取上海黄金交易所现货合约基础信息。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 合约代码（支持多个） |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 品种代码 |
| ts_name | str | 品种名称 |
| trade_type | str | 交易类型 |
| t_unit | float | 交易单位（克/手） |
| p_unit | float | 报价单位 |
| min_change | float | 最小变动价位 |
| price_limit | float | 每日价格最大波动限制 |
| min_vol | int | 最小单笔报价量（手） |
| max_vol | int | 最大单笔报价量（手） |
| trade_mode | str | 交易期限 |
| margin_rate | float | 保证金比例 |
| liq_rate | float | 违约金比例（%） |
| trade_time | str | 交易时间 |
| list_date | str | 上市日期 |

## 代码示例

```python
import tushare as ts
pro = ts.pro_api('your_token')
df = pro.sge_basic()
print(df.head())
```
