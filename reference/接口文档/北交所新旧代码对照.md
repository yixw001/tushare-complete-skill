## 北交所新旧代码对照表

接口：bse_mapping描述：获取北交所股票代码变更后新旧代码映射表数据限量：单次最大1000条（本接口总数据量300以内）积分：120积分即可调取

输入参数

<table>
<thead>
<tr>
<th>名称</th>
<th>类型</th>
<th>必选</th>
<th>描述</th>
</tr>
</thead>
<tbody><tr>
<td>o_code</td>
<td>str</td>
<td>N</td>
<td>旧代码</td>
</tr>
<tr>
<td>n_code</td>
<td>str</td>
<td>N</td>
<td>新代码</td>
</tr>
</tbody></table>
输出参数

<table>
<thead>
<tr>
<th>名称</th>
<th>类型</th>
<th>默认显示</th>
<th>描述</th>
</tr>
</thead>
<tbody><tr>
<td>name</td>
<td>str</td>
<td>Y</td>
<td>股票名称</td>
</tr>
<tr>
<td>o_code</td>
<td>str</td>
<td>Y</td>
<td>原代码</td>
</tr>
<tr>
<td>n_code</td>
<td>str</td>
<td>Y</td>
<td>新代码</td>
</tr>
<tr>
<td>list_date</td>
<td>str</td>
<td>Y</td>
<td>上市日期</td>
</tr>
</tbody></table>
接口示例

```

#获取方大新材新旧代码对照数据
df = pro.bse_mapping(o_code='838163.BJ')

#获取全部变更的股票代码对照表
df = pro.bse_mapping()

```

数据示例

```
      name     o_code   n_code    list_date
0  方大新材  838163.BJ  920163.BJ  20200727

```
