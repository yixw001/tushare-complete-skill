## 期货合约信息表

接口：fut_basic描述：获取期货合约列表数据限量：单次最大10000积分：用户需要至少2000积分才可以调取，具体请参阅积分获取办法

接口示例

```

pro = ts.pro_api()

#获取郑商所所有合约列表
df = pro.fut_basic(exchange='CZCE', fields='ts_code,symbol,exchange,name,fut_code,list_date,delist_date')

#获取大商所2020年上市的所有合约列表
df = pro.fut_basic(exchange='DCE', list_date='20200101')

#获取白银合约信息
df = pro.fut_basic(exchange='SHFE', fut_code='AG')

#获取中金所股指期货主力合约
df = pro.fut_basic(exchange='CFFEX', fut_type='2')

```
