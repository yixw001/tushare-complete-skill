# 指数日线行情

**文档ID**: 76
**接口**: index_daily

## 接口描述

获取指数日线行情。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 指数代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期(YYYYMMDD) |
| end_date | str | N | 结束日期(YYYYMMDD) |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 指数代码 |
| trade_date | str | 交易日期 |
| close | float | 收盘点数 |
| open | float | 开盘点数 |
| high | float | 最高点数 |
| low | float | 最低点数 |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.index_daily(
    ts_code='000001.SH',
    start_date='20240101',
    end_date='20241231'
)
print(df.head())
```
