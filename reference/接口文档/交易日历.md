# 交易日历

**文档ID**: 137
**接口**: trade_cal

## 接口描述

获取各大交易所交易日历数据。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| exchange | str | N | 交易所：SSE上交所，SZSE深交所，CFFEX中金所，SHFE上期所，DCE大商所，CZCE郑商所，INE上国际能源 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| is_open | int | N | 是否交易 0休市 1交易 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| exchange | str | 交易所代码 |
| cal_date | str | 日历日期 |
| is_open | int | 是否开市 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.trade_cal(
    exchange='SSE',
    start_date='20240101',
    end_date='20241231'
)
print(df.head())
```
