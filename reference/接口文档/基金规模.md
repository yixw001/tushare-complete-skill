## 基金规模数据

接口：fund_share，可以通过数据工具调试和查看数据。描述：获取基金规模数据，包含上海和深圳ETF基金限量：单次最大提取2000行数据积分：用户需要至少2000积分可以调取，5000积分以上频次较高，具体请参阅积分获取办法

代码示例

```
#初始接口
pro = ts.pro_api()

#单只基金
df = pro.fund_share(ts_code='150018.SZ')

#多只基金
df = pro.fund_share(ts_code='150018.SZ,150008.SZ')

```
