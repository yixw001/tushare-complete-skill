## 台湾电子产业月营收

接口：taiwan_revenue描述：获取台湾电子产业月营收数据限量：单次最大2000，可多次提取，总量不限制积分：用户需要至少2000积分才可以调取，5000积分以上频次相对较高，积分越多权限越大，具体请参阅积分获取办法

接口示例

```

import tushare as ts

pro = ts.pro_api()

#获取台湾电子产业月营收数据
df = pro.taiwan_revenue(start_date='20200101', end_date='20201231', fields='ts_code,trade_date,revenue,yoy,mom')

```
