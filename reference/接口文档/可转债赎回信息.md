## 可转债赎回信息

接口：cb_call描述：获取可转债赎回信息限量：单次最大2000，可多次提取，总量不限制积分：用户需要至少2000积分才可以调取，5000积分以上频次相对较高，积分越多权限越大，具体请参阅积分获取办法

接口示例

```

import tushare as ts

pro = ts.pro_api()

#获取可转债赎回信息
df = pro.cb_call(start_date='20190101', end_date='20201231', fields='ts_code,ann_date,call_price,call_area,call_records')

```
