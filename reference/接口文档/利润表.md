# 利润表

**文档ID**: 32
**接口**: income

## 接口描述

获取上市公司利润表数据。用户需要至少2000积分才可以调取。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| start_date | str | N | 公告日开始日期 |
| end_date | str | N | 公告日结束日期 |
| period | str | N | 报告期 |
| comp_type | str | N | 公司类型：1一般工商业 2银行 3保险 4证券 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| total_revenue | float | 营业总收入 |
| revenue | float | 营业收入 |
| n_income | float | 净利润 |
| n_income_attr_p | float | 归属于母公司所有者的净利润 |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.income(
    ts_code='600000.SH',
    start_date='20240101',
    end_date='20241231'
)

print(df[['ts_code', 'end_date', 'total_revenue', 'revenue', 'n_income_attr_p']].head())
```

## 重要字段说明

- **n_income_attr_p**：归属于母公司所有者的净利润，是评估公司盈利能力的核心指标
