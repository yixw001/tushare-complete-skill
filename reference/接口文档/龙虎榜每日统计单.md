# 龙虎榜每日统计单

**文档ID**: 52
**接口**: top_list

## 接口描述

获取龙虎榜每日统计数据。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| exchange_id | str | N | 交易所代码 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| close | float | 收盘价 |
| pct_change | float | 涨跌幅（%） |
| amount | float | 换手率（%） |
| l_sell | float | 龙虎榜卖出金额（万元） |
| l_buy | float | 龙虎榜买入金额（万元） |

## 代码示例

```python
import tushare as ts

pro = ts.pro_api('your_token')

df = pro.top_list(trade_date='20241231')
print(df.head())
```
