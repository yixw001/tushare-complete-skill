## 债券回购日行情

接口：repo_daily描述：债券回购日行情限量：单次最大2000条，可多次提取，总量不限制权限：用户需要累积2000积分才可以调取，具体请参阅积分获取办法

```

df = pro.repo_daily(trade_date='20200804')


```
