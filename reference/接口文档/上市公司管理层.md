# 上市公司管理层

**文档ID**: 193
**接口**: stk_managers

## 接口描述

获取上市公司管理层信息。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码（支持多个） |
| ann_date | str | N | 公告日期（YYYYMMDD） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 输出参数

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| name | str | 姓名 |
| gender | str | 性别 |
| lev | str | 岗位类别 |
| title | str | 岗位 |
| edu | str | 学历 |
| national | str | 国籍 |
| birthday | str | 出生年月 |
| begin_date | str | 上任日期 |
| end_date | str | 离任日期 |

## 代码示例

```python
import tushare as ts
pro = ts.pro_api('your_token')
df = pro.stk_managers(ts_code='000001.SZ')
print(df.head())
```
