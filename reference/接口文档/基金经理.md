## 基金经理

接口：fund_manager描述：获取公募基金经理数据，包括基金经理简历等数据限量：单次最大5000，支持分页提取数据积分：用户有500积分可获取数据，2000积分以上可以提高访问频次

代码示例

```
#初始接口
pro = ts.pro_api()

#单只基金
df = pro.fund_manager(ts_code='150018.SZ')

#多只基金
df = pro.fund_manager(ts_code='150018.SZ,150008.SZ')

```
