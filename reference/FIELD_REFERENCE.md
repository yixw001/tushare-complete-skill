## 重要字段说明

### 在建工程 (cip)

**重要性**: ⭐⭐⭐

在建工程是公司正在建设的工程项目价值，反映公司未来产能扩张的投入。

**获取方式**:
```python
df = pro.balancesheet(ts_code='600000.SH', start_date='20240101', end_date='20241231')
print(df['cip'])
```

### 资本开支 (c_pay_acq_const_fiolta)

**重要性**: ⭐⭐⭐

资本开支是公司购建固定资产、无形资产和其他长期资产支付的现金，反映公司再投资能力。

**获取方式**:
```python
df = pro.cashflow(ts_code='600000.SH', start_date='20240101', end_date='20241231')
print(df['c_pay_acq_const_fiolta'])
```

### 扣非净利润 (n_income_attr_p)

**重要性**: ⭐⭐

扣非净利润是归属于母公司所有者的净利润，反映公司核心盈利能力。

**获取方式**:
```python
df = pro.income(ts_code='600000.SH', start_date='20240101', end_date='20241231')
print(df['n_income_attr_p'])
```

### 净资产收益率 (roe)

**重要性**: ⭐⭐

ROE是衡量公司盈利能力的重要指标。

**获取方式**:
```python
df = pro.fina_indicator(ts_code='600000.SH', start_date='20240101', end_date='20241231')
print(df['roe'])
```
